# Sales-Trend-Analysis-Task-6
This project analyzes sales data using SQL to identify monthly revenue trends and order performance. The objective is to extract meaningful business insights from raw sales records by applying aggregation, filtering, and sorting techniques. To Analyse monthly revenue and order volume.
The dataset used is "100 sales records."
Tasks performed:
1. Extracted Year and Month from the Order Date field.
2. Filtered data for:
    Months: March, April, May, and June
    Years: 2012 to 2014
3. Grouped data by Year and Month.
4. Calculated total orders(Count of distinct Order IDs), monthly revenue(SUM of Total Revenue), avergae order value(revenue per order)
5. Sorted results by Year and Month.
6. **Output**
7. The final results table consists of: Year, month, monthly_revenue, total_orders, and avg_order_value.
